Ejercicio 8 - Francisco Ramírez

Mediante un ejemplo real, indica la utilidad del método exec();

let expresion = /a/;

let string = "hola, me gusta el azul";

expresion.exec(string);

Salida: ["a", index: 3, input: "hola, me gusta el azul", groups: undefined]

Este método devuelve el texto coincidente si encuentra una coincidencia; de lo contrario, devuelve un valor nulo.
