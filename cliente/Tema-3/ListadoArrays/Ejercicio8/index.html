Ejercicio 8 - Francisco Ramírez

Indica la diferencia entre los siguientes métodos, y demuestra su uso con algunos arrays: Array.prototype.forEach(), Array.prototype.every(), Array.prototype.some() y Array.prototype.filter()

-------------------------------Teoría--------------------------

Array.prototype.forEach()

Recorre el array elemento a elemento
  1. array = [0,1,2,3, "hola"];
  2. Usamos el forEach de la siguiente forma:
    array.forEach(elemento, indice, array)
    ("Array[" + indice + "] = " + elemento);

Array.prototype.every()

Comprueba si todos los elementos del array pasando una prueba para devolver true en caso afirmativoo false en caso negativo.
Su uso sería el siguiente:
  1. array = [0,1,2,3, "hola"];
  2. Usamos el every de la siguiente forma:
    (array.every(function(numero){return !isNaN(numero);}));
    false, existe un string

Array.prototype.some()

Es parecido al every, pero comprueba si hay algún elemento que cumpla una condición. En el momento que haya uno que lo cumpla, ya devuelve true, y false en caso negative. Su uso sería el siguiente:
  1. array = [0,1,2,3 "hola"];
  2. Usamos el some de la siguiente forma:
    (array.some(function(numero){return numero;}));
    true, hay elementos que si lo son.

Array.prototype.filter()

Filtra elementos de un array y se guardar en un nuevo array.
  1. array = [0,1,2,3, "hola"];
  2. Usamos el filter de la siguiente forma:
    array.filter(function(numero){numero < 4;}));
