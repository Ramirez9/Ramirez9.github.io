Animaciones - Francisco

Guía sobre las animaciones con CSS3

¿Qué es una animación?

Son movimientos entre los estilos. Ventajas de las nuevas animaciones CSS3:

1.-Se puede emplear sin JavaScript
2.-No pesan demasiado, los equipos no potentes las pueden soportar.
3.-Permite optimizar el rendimiento y eficiencia de la animación dependiendo el navegador.

Declarar animaciones

Para declarar una animación necesitamos hacer uso de la regla reservada @keyframe

Esta regla define los fotogramas de dicha animación. Existen dos formas:

1.-Forma

@keyframe nombreRegla{
from {left: 0px} ----> Primer fotograma
to {left: 900px} ----> Último fotograma

2.-Forma

@keyframe nombreRegla{
0% {left: 0px} ---> Primer fotograma
50% {left: 900px; background:#8080c0} ----> Segundo fotograma
100% {left: 0px} ----> Último fotograma

La segunda forma es más detallada cuando necesitamos de más selectores. Realmente from y to se traducen por 0% y 100%

Animation-name

Se encargar de especificar el nombre para la animación
Obviamente esta propiedad no funciona a menos que hagamos uso de las siguientes propiedas explicadas

animation-name: nombreRegla;

Animation-duration

Indica la duración de la animación, se mide en segundos y su efecto por valor es 0s

animation-duration : tiempo
animation-duration : 1s
animation-duration : 5s
animation-duration : 10s
#animation-duration-10 {
width: 100px;
height: 50px;
background: rgb(209, 86, 86);
position: relative;
animation: animation-duration infinite;
animation-duration: 10s;
}

@keyframes animation-duration {
from {left: 0px;}
to {left: 800px;}
}

Animation-timing-function

Especifica como la animación debe avanzar según la duración del ciclo

Valores:
  ease;
  ease-in;
  ease-out;
  ease-in-out;

  linear;
  step-start;
  step-end;

  cubic-bezier(P0, P1, P2, P3);
    Con cubic-bezier podemos representar los demás valores anteriores.Ej:
    linear = cubic-bezier(0, 0, 1, 1);

animation-timing-function: valor;
animation-timing-function: ease;Inicio lento - rápido - acaba lentamente
animation-timing-function: ease-in;Inicio lento
animation-timing-function: ease-out;Final lento
animation-timing-function: ease-in-out;Inicio lento y final rápido
animation-timing-function: linear;Inicio = Final
animation-timing-function: step-start;Se detiene al final
animation-timing-function: step-end;Se detiene al principio
animation-timing-function: cubic-bezier(0, 0, 1, 1) = linear;Define una curva

#animation-timing-function-cubic-bezier{
width: 120px;
height: 50px;
background: rgb(121, 223, 126);
position: relative;
animation: animation-timing-function 10s infinite;
animation-timing-function: cubic-bezier(0, 0, 1, 1);
}

@keyframes animation-timing-function {
from {left: 0px;}
to {left: 600px;}
}

Animation-delay

Especifica el tiempo de retardo que debe transcurrir antes de comenzar la animación.
Esto permite que la secuencia de animación no comience nada más ser aplicada a un elemento.
Su valor por defecto es 0, a menos que indiquemos alguno en segundos o milisegundos

Animation-delay: tiempo
Animation-delay: 2s
Animation-delay: 3s
Animation-delay: 6s
#animation-delay-15{
width: 140px;
height: 50px;
background: rgb(137, 96, 156);
position: relative;
animation: animation-delay 10s infinite;
animation-delay: 15s;
}

@keyframes animation-delay {
0% {left: 0px;}
50% {left: 50px; background: indigo}
100% {left: 600px;}
}

Animation-iteration-count

Define el dúmero de veces que un ciclo debe de ejecutarse.
Existen dos valores:
  infinite, usado en ejemplos anteriores
  Number, valor por defecto 1, no admite -1 y admite decimales en ciclos

animation-iteration-count: infinite | number
animation-iteration-count: infinite;Se repite infinitamente
animation-iteration-count: 2;Se repite 2 veces
animation-iteration-count: 10;Se repite 10 veces
#animation-iteration-count-10{
width: 140px;
height: 50px;
background: rgb(235, 200, 87);
position: relative;
animation: animation-iteration 5s;
animation-iteration-count: 10;
}

@keyframes animation-iteration {
0% {left: 0px;}
100% {left: 900px; background: rgb(255, 21, 0)}
}

Animation-direction

Define si una animación se debe reproducir hacia adelante, hacia atrás o en ciclos alternos.
Su valor por defecto es normal.

Existen 4 valores:
  normal
  alternate
  reverse
  alternate-reverse

animation-direction: valor;
animation-direction: normal;Ciclo normal
animation-direction: alternate;De izquierda a derecha y viceversa
animation-direction: reverse;De derecha a izquierda
animation-direction: alternate-reverse;De derecha a izquierda y viceversa
#animation-direction-alternate-reverse{
width: 140px;
height: 50px;
background: rgb(129, 212, 233);
position: relative;
animation: animation-direction 5s infinite;
animation-direction: alternate-reverse;
}

@keyframes animation-direction {
0% {left: 0px;}
100% {left: 900px; background: rgb(38, 0, 255)}
}

Animation-fill-mode

Especifica un estilo para el elemento cuando la animación no se está reproduciendo. Su valor por defecto es none.

Existen 4 valores:
  none
  forwards
  backwards
  both

animation-fill-mode: valor;

Recargar para verlo detalladamente

animation-fill-mode: none;
animation-fill-mode: forwards;
animation-fill-mode: backwards;
animation-fill-mode: both;
#animation-fill-mode-both{
width: 140px;
height: 50px;
position: relative;
animation-delay: 1s;
animation: animation-fill-mode 5s ;
animation-fill-mode: both;
animation-delay: 3s;
}

@keyframes animation-fill-mode {
from {left: 0px; background: rgb(17, 0, 0)}
to {left: 900px; background: rgb(255, 0, 212)} }

Animation-play-state

Determina si está en ejecución o en pausa

Existen 2 valores:
  running
  paused

animation-play-state: valor;
animation-play-state: running;Coloca el puntero encima
animation-play-state: paused;Coloca el puntero encima
#animation-play-state-paused{
width: 300px;
height: 50px;
background: #90A4AE;
position: relative;
animation: animation-play-state 5s infinite;
}

#animation-play-state-paused:hover{
animation-play-state: paused;

@keyframes animation-play-state {
from {left: 0px;}
to {left: 900px;}
}

Animation

Es una abreviación de las propiedades explicadas.
Sus valores por defecto son:
  animation-name: none
  animation-duration: 0s
  animation-timing-function: ease
  animation-delay: 0s
  animation-iteration-count: 1
  animation-direction: normal
  animation-fill-mode: none
  animation-play-state: running

animation : "name" "duration" "timing-function" "delay" "iteration-count" "direction" "fill-mode"

animation: animation-name 10s ease-in-out 5s infinite alternate forwards;
